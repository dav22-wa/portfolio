## TROUBLESHOOTING

### If Images Don't Load
- Check file paths are correct
- Verify image files exist in assets folder
- Check file extensions match (jpg vs jpeg)
- Ensure images are not too large (compress if >500KB)

### If Styles Look Broken
- Clear browser cache (Ctrl+Shift+R)
- Check both CSS files are linked
- Verify no CSS syntax errors
- Test in incognito mode

### If Forms Don't Work
- Check JavaScript console for errors
- Verify form field names match
- Test with simple email setup
- Consider using Formspree or similar service

### If Site is Slow
- Compress images (use TinyPNG.com)
- Minify CSS and JavaScript
- Enable caching
- Use CDN for libraries
